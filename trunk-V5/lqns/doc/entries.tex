\chapter{Entries}
\label{sec:entries}

Connections between \link{entities}{sec:entities} in the layered
queueing network model are actually from an
\link{\dfn{entry}}[~(\Sec\Ref)]{sec:entry} on the source entity to an
entry on the destination entry.  Connections between entries are made
through objects of class \link{\file{Call}}[~(\Sec\Ref)]{sec:call}.
The class relationships for entries and calls are shown in
\htmlonly{the \link{figure
    below}{fig:entry}}\texonly{Figure~\ref{fig:entry}}.

\end{figure}
